# Dark Minimal Premium Study Mode

Use only tokens from `tokens.json` for the flashcards block.

## Rules
- No gradients.
- No glow effects.
- No glassmorphism.
- Radius: cards 28, buttons 16, chips 999, tiles 14.
- Spacing only from 8-point scale.
- Typography only from defined scale and weights.
- Shadows only from token set.
